#  E-Commerce Project

A full-stack E-Commerce web application built with **Next.js (Frontend)** and **Express.js (Backend)**.  
This project provides a complete setup for user authentication, product management, and order handling.

---

##  Setup Instructions

 **Clone the repository and install dependencies**

```bash
git clone https://github.com/username/E-Commerce.git
cd E-Commerce



## Frontend terminal
    cd frontend
    npm install


## Backend terminal
    cd backend
    npm install


2️⃣ Setup environment variables

    Inside the backend folder, create a file named .env
    and copy the content from .env.example, then fill in your actual values


    Backend:
        npm run dev
    
    Frontend:
        npm run dev


    Backend runs on → http://localhost:5000
    Frontend runs on → http://localhost:3000
